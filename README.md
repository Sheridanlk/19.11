- Были использованы сторонние библиотеки для загрузки конфига(cleanenv) и для генерации PDF(go-pdf/fpdf)

- Под перезагрузской сервера я понял его аккрутаное завершение(graceful shutdown), при котором текущие задачи завершаются, данные сохраняются на диск, при следующем запуске загружаются обратно.

- Для запуска нужно задать абсолютну ссылку на конфиг в переменную окужения "CONFIG_PATH"
